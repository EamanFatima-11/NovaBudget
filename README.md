💚 NovaBudget — Personal Finance Tracker

Smart money. Smarter you.

A fully functional Flutter personal finance app with real-time balance tracking, custom budget limits, live spending charts, and complete add/edit/delete functionality — all in a stunning deep emerald theme.

✨ Features

💰 Live Balance Card — Total balance, income, expenses auto-calculated
➕ Add Transactions — Income or expense with 8 categories
🗑️ Swipe to Delete — Swipe left on any transaction with undo option
📊 Set Budget Limits — Monthly limits per spending category
✏️ Edit Budgets — Tap any budget card to update the limit
⚠️ Over-limit Warning — Red alert when spending reaches 90% of limit
📈 Weekly Bar Chart — Live spending per day for last 7 days
🍩 Category Donut Chart — Visual breakdown by spending category
💡 Savings Rate — Live percentage of income being saved
🧾 Stats Grid — Balance, income, expenses, transaction count

🎯 How It Works
1. App loads with sample transactions and budgets
2. Tap "Add Transaction" → enter title, amount, category
3. Swipe left on any transaction → delete with undo
4. Tap bank icon → set a new budget limit
5. Tap any budget card → edit the limit
6. Overview tab → see live charts and stats
